# FileFlow - File Management System (2026)

A modern, responsive file management system built with Flask and Bootstrap.

## Features
- 📁 Upload files (drag & drop support)
- 🔍 Search and filter files
- 📊 Statistics dashboard
- 🖼️ File type icons
- 📱 Responsive design
- ⌨️ Keyboard shortcuts

## Installation
1. Clone the repository
2. Create virtual environment: `python -m venv venv`
3. Activate it and install: `pip install -r requirements.txt`
4. Run: `python app.py`

## Usage
- Visit `http://localhost:5000`
- Press `Ctrl+U` to upload files
- Click on files to download
- Use search bar to find files
